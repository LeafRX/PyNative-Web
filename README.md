# PyNative-Web

**EN:** A zero-dependency asynchronous web framework built from scratch  
**RU:** Минималистичный асинхронный веб-фреймворк без зависимостей

### Requirements:
* **Python:** 3.8+
* **OS:** Linux, Windows
* **Dependencies:** None

### Setup:
#### Clone repository
```bash
git clone https://github.com/leafrx/pynative-web.git
cd pynative-web

python -m venv venv
source venv/bin/activate 

# Windows:
# .\venv\Scripts\activate

# Start server:
python app.py
